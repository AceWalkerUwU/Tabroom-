# Privacy Policy for Tabroom+

**Last updated:** February 1, 2026

**DISCLAIMER**
The parli predicting pairings feature is NOT 100% accurate, please contact if there are grave issues (i.e. its saying you're going to lose even when it's clearly not the case as it will mix up or mess up names making it sometimes innacurate)

Tabroom+ is a Google Chrome extension that enhances the Tabroom website with tools to help debate participants track tournaments, save preparation notes, and view informational enhancements.

1) Data Collection
Tabroom+ stores limited user-generated data, including:
- Starred tournaments
- Personal notes on judges and competitors

This data is stored using Chrome’s storage APIs (local and sync) to allow it to persist across browser sessions and, if enabled by the user, across signed-in Chrome profiles.

2) Data Usage
- All stored data is created voluntarily by the user
- Data is used only to provide the extension’s functionality
- Tabroom+ does not collect analytics, track users, or monitor browsing activity

3) Data Sharing
Tabroom+ does **not** sell, share, or transmit user data to third parties.

The extension may fetch publicly available data (such as rankings in CSV format from the NPDL Ranking google sheets: https://docs.google.com/spreadsheets/d/1oz6E9Bxw7d__DmNWJykS3VcRvJivffX7y_Jqtw7YxcU/edit?usp=sharing) for informational purposes. 
This data does not contain personal user information and is not linked to individual users.

4) Data Security
All user data remains within the user’s browser environment and is not accessible to the developer or any third party.
